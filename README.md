# SearchEngine
-Adds title, season, and episode number
-Sorts by descending number of words in each script that match the query, splits multi word queries.  Result with most words matched will be top result.
-Makes the view button functional when clicking on a result.  This takes the user to the full script of the episode
-Highlights matched query/queries when viewing scripts, for example if you type in "beer homer" both instances of beer and homer will be highlighted
-Made privacy policy
